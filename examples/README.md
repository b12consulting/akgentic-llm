# akgentic-llm Examples

This directory will contain working examples demonstrating akgentic-llm features.

## Planned Examples

1. **01_hello_llm.py** - Simple prompt/response with multiple providers
2. **02_context_management.py** - Checkpoint, rewind, and compactification
3. **03_usage_limits.py** - Cost control with usage limits
4. **04_dynamic_prompts.py** - System prompt registry usage
5. **05_tool_integration.py** - Agent with custom tools

Examples will be implemented during Sprint 3 of the akgentic-llm development cycle.
